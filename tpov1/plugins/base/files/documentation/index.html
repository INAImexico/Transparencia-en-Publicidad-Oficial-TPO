How to use Media Manager

This is ready to use script for your codeigniter projects. In order to use media manager files in your project, you need to merge model, view and controller files into your project's mvc folder. Media manager inclues following files:

ControllersModalsViews
media.phpmanager.phpindex.php
manager.php
medialist.php

You can access your project's MVC folders (models, views and controllers) in:
your-project-folder/application/...

Media Manager uses folder named "assets" placed in root folder as base folder for calling all its css, js, fonts and other files.

  1. First place media.php file into your controllers folder.
    On line no. 23 change the location of model file.
    $this->load->model('your-models-path-here/manager');

    On line no. 87 change the location of view files from where you want to load media manager view files.
    $this->load->view('your-views-path-here/index',$data);

    You can place these view files into a folder (like folder named "mediamanager"). Then you have to change the above line to:--
    $this->load->view('mediamanager/index',$data);

    'index.php' file calls required css and js files, and loads layout of media manager . So if you want, you can merge it with your own index.php file (in case you have knowledge of working with html and php)
  2. Place model file 'manager.php' into your models folder.
  3. Place view files of media manager into your views folder.
  4. After placing all files in place you need to create a link to call media manager in your project's view file. For eg:-
    If you have placed controller file 'media.php' into controllers/admin folder, then create link to it as-
    <a href="<?php site_url('admin/media'); ?>">Media Manager</a>

HTML Structure

The HTML structure is a fluid layout with two columns. The folder tree structure content is nested within a div with an id of "treeview". The media container (column #2) content is within a div with an id of "mediaview". The media container element lies in two div with ids "thumbsView" and "detailsView". "thumbsView" contents thumbnail view of media while "detailsView" contents tabular structure of media. The general template structure is the same throughout the template. Javascript

This theme imports three Javascript files.

  1. bootbox.min.js
  2. bootstrap.min.js
  3. client.js
  4. cookie.js
  5. dropzone.min.js
  6. html5shiv.min.js
  7. jquery.magnific-popup.min.js
  8. jquery-1.11.1.min.js
  9. masonry.pkgd.min.js
  10. media.js
  11. respond.min.js

  1. Bootbox is a jquery plugin which uses bootstrap to show alert messages in bootstrap modals.
  2. Bootstrap is a popular html, css and js framework for developing responsive websites.
  3. client.js collects client side browser information like operating system, browser etc to support cross browser compatibility for file upload.
  4. jquery cookie plugin is to save current view (thumb or details) on user's system.
  5. dropzone supports drag and drop feature for file upload.
  6. HTML5Shiv is a JavaScript workaround, invented by Sjoerd Visscher, to enable styling of HTML5 elements in versions of Internet Explorer prior to version 9, which do not allow unknown elements to be styled without JavaScript.
  7. Magnific Popup is a responsive lightbox & dialog script to show image popups.
  8. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  9. Masonry is a JavaScript grid layout library. It works by placing elements in optimal position based on available vertical space
  10. Most of the workarounds of media manager uses media js script. There are a few functions worth looking over. 

    Initialize dropzone code to support drag and drop.

Media Manager Settings

  • Allowed Types: Comma separated list of file extensions allowed to upload via script.
    Add your own extensions separated by comma (it must be supported by CodeIgniter).
  • Maximum Size: Maximum file size (in MB) allowed to upload, set 0 for no limit.
  • Maximum Width: Maximum width of image (in pixels) allowed to upload, set 0 for no limit.
  • Maximum Height: Maximum height of image (in pixels) allowed to upload, set 0 for no limit.
  • Media Folder: Root media folder to show directory structure and media
  • Maximum File Name: Maximum no. of characters a uploaded file can have, set 0 for no limit.
  • Overwrite: Overwrite files with same name while uploading.
  • Remove Spaces: Remove spaces from names of uploaded files.
  • Encrypt File Name: Encrypt upload file name
  • Restore Default Settings: Restore default settings of media manager.
